## Spacing Scale

Declared values (all multiples of 4). Tailwind v4 spacing tokens map directly.

| Token | Value | Tailwind class example | Usage in Phase 2 |
|-------|-------|------------------------|------------------|
| xs | 4px | `gap-1` | Icon-to-label gaps, badge offsets |
| sm | 8px | `gap-2` / `p-2` | Product card inner padding, button gaps |
| md | 16px | `gap-4` / `p-4` | Default element spacing, grid gaps (mobile) |
| lg | 24px | `gap-6` / `px-6` | Section padding, PDP info panel padding |
| xl | 32px | `gap-8` / `py-8` | Layout column gaps, shop page vertical rhythm |
| 2xl | 48px | `py-12` | Major section breaks (PDP gallery → info) |
| 3xl | 64px | `py-16` | Page-level top/bottom padding |

**Exceptions:**
- Touch targets: minimum 44px × 44px on all interactive elements — already enforced in globals.css for `a, button, [role="button"]`
- Size selector buttons: minimum 44px × 44px touch target, regardless of label width
- Cart badge: 20px × 20px pill, positioned at `top-0 inline-end-0` on the cart icon using logical properties
- CartBadge overlap offset: −6px — decorative positioning micro-offset for badge-on-icon overlap; not a layout spacing token.
- Navbar height: 64px (h-16) — established in Phase 1, unchanged

## Typography

Font switches automatically via `:lang()` CSS selector — no component className needed.
Arabic body text renders at 17px (1.0625rem) to match 16px Latin visual weight — enforced in globals.css.

| Role | EN Size | AR Size | Weight | Line Height | Letter Spacing | Usage |
|------|---------|---------|--------|-------------|----------------|-------|
| Body | 16px (1rem) | 17px (1.0625rem) | 400 (normal) | 1.6 | 0 | Product descriptions, filter labels, nav links, policy snippet |
| Label / UI | 14px (0.875rem) | 14px (0.875rem) | 400 (normal) | 1.4 | 0 | Price labels, "12 Products" count text, size button labels, badge text, sort dropdown |
| Heading | 20px (1.25rem) | 20px (1.25rem) | 600 (semibold) | 1.3 | −0.02em | Product name on PDP info panel, category page heading, size guide modal title |
| Display | 28px (1.75rem) | 28px (1.75rem) | 600 (semibold) | 1.2 | −0.02em | Shop page hero heading ("Tops", "Bottoms", "All Products") |

**Weights used: exactly 2 — 400 (normal) and 600 (semibold).**

**Product card name:** 16px, weight 400, truncated to 1 line on mobile / 2 lines on desktop. On hover: `text-decoration: underline` fires simultaneously with image zoom (D-09).

**Exceptions:**
- 12px — CartBadge numeral and StockStateBadge card pill: fixed-size container constraints, not typographic choices. Both elements are decorative micro-labels inside fixed-height pill containers.

## Color

Palette: black and white only — no other hues. Established in globals.css ORKI custom properties.

| Role | Hex / Token | CSS Variable | Usage |
|------|-------------|--------------|-------|
| Dominant (60%) | `#000000` | `--color-dominant` | Page background, Navbar background, PDP background |
| Secondary (30%) | `#111111` | `--color-secondary-surface` | Product card background, category tab bar background, sort dropdown panel, size guide modal background |
| Accent (10%) | `#ffffff` | `--color-accent-orki` | See reserved list below |
| Border | `rgba(255,255,255,0.12)` | `--color-orki-border` | Navbar bottom border, card borders, modal borders, size selector button borders |
| Placeholder bg | `#0A0A0A` | `--color-placeholder-bg` | All PlaceholderImage dark-field backgrounds |
| Placeholder mark | `rgba(255,255,255,0.15)` | `--color-placeholder-mark` | Ghost ORKI wordmark inside PlaceholderImage |
| Muted text | `rgba(255,255,255,0.60)` | (inline opacity) | Nav link default state, filter tab inactive, sort label, product category tag |
| Destructive | `oklch(0.704 0.191 22.216)` | `--destructive` (dark mode) | Fully out-of-stock state only — see StockStateBadge |

**Accent (`#ffffff`) is reserved for exactly these elements — nowhere else:**
1. Primary CTA button ("Add to Cart") — white background, black text
2. Active category tab indicator (underline or full background depending on tab style)
3. Size selector button — selected state (white bg, black text)
4. Cart badge background (white pill, black numeral)
5. Product name underline on card hover (white underline)
6. Success state checkmark icon inside AddToCartButton (1.5s window)
7. StockStateBadge "In Stock" text / indicator dot

## Animation Contract

Extensions to `src/lib/animation-presets.ts` required for Phase 2. The executor MUST add these presets to the existing file — never use ad-hoc values.

```ts
// ADD to animationPresets in src/lib/animation-presets.ts:

cardHover: {
  // Product card image zoom — 300ms ease-out (smooth, not snappy)
  duration: 0.3,
  ease: [0, 0, 0.2, 1] as const,   // CSS ease-out equivalent
},
badgePop: {
  // Cart badge first-appear scale pop — 150ms spring-like
  duration: 0.15,
  ease: [0.34, 1.56, 0.64, 1] as const,  // Slight overshoot for tactile feel
},
dropdownOpen: {
  // CategoryDropdown fade + slide — 150ms ease
  duration: 0.15,
  ease: [0.23, 1, 0.32, 1] as const,   // reuse navEnter easing
},
successState: {
  // AddToCartButton label swap fade — 150ms
  duration: 0.15,
  ease: [0.23, 1, 0.32, 1] as const,
},
```
