**If lint fails with physical property violations:**
- Find the offending file via the ESLint error message
- Replace the physical class with its logical equivalent:
  - `ml-*` → `ms-*`
  - `mr-*` → `me-*`
  - `pl-*` → `ps-*`
  - `pr-*` → `pe-*`
  - `left-*` (position) → `inset-inline-start-*`
  - `right-*` (position) → `inset-inline-end-*`

**If build fails with type errors on searchParams/params:**
- Ensure the page component is `async` and both `params` and `searchParams` are awaited
- Check that the type is `Promise<{...}>` not `{...}` directly
